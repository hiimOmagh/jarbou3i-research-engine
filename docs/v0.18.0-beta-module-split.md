# v0.18.0-beta — Research Engine Module Split

v0.18.0-beta is a behavior-preserving architecture release. It reduces the central weight of `src/research-engine.js` by moving stable boundaries into focused browser modules while keeping manual/private mode, provider flows, migration, privacy export, and evidence-review behavior unchanged.

```text
src/research/evidence-review-controller.js
src/research/provider-controller.js
```


## Boundary intent

- `render-helpers.js`: translation copy, HTML escaping, label/placeholder rendering helpers.
- `state-store.js`: default state, local persistence, state hydration, evidence renumbering.
- `evidence-controller.js`: evidence IDs, score clamping, model-layer ID helpers.
- `evidence-review-controller.js`: pending review selection and queue metrics.
- `provider-controller.js`: provider endpoint defaults and provider config sanitation boundary.
- `source-controller.js`: source policy boundary over source connector contracts.
- `export-controller.js`: JSON download path and privacy-export guard routing.
- `quality-gate.js`: Quality Gate v2 scoring logic.

## Guardrails

- No feature expansion in this release.
- No real OAuth.
- No real source crawling.
- No behavior change intended.
- Privacy-safe export remains the only JSON download path.
- Migration-first import remains active.
- Mock/manual mode remains first-class.
