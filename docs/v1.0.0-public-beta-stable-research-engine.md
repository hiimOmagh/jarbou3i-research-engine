# v1.0.0 — Public Beta / Stable Research Engine

## Release basis

v1.0.0 is a promotion from v0.29.0-rc.1 after external CI/browser validation was reported successful.

This release introduces no new feature surface. It is a stability/versioning promotion only.

## Stable baseline

Included and frozen as the public beta baseline:

- manual/private research workflow
- provider mode abstraction and validation
- privacy-safe export guard
- privacy audit release gate
- migration compatibility from v0.11 through v0.29.0-rc.1
- project workspace/local storage management
- analysis template system
- Advanced Quality Gate v3
- Export Pack v2
- backend provider hardening
- GitHub public source connector prototype
- web search provider abstraction, dry-run only
- portable OAuth PKCE spike, not production OAuth
- release/stable metadata

## Still intentionally not enabled

- production OAuth token vault
- live web search
- new source connectors beyond the GitHub public metadata prototype
- direct source-result promotion into Evidence Matrix
- browser-exported provider/API/OAuth secrets

## Release gates

Before tagging v1.0.0, the required external gate is:

```bash
npm install --no-audit --no-fund
npm run test:ci:no-browser
npm run test:ci:browser
```

The user reported the new CI succeeded before this promotion package was created.
