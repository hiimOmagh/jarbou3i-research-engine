# XR-13.0.3 — Guided Workflow Contract Token Hotfix

## Purpose

Unblock the XR-13 user-friendly product experience recomposition after the browser suite passed but the no-browser static gate failed on a literal guided-workflow token.

## Failure

The runtime/browser behavior was green, but `npm run test:ci:no-browser` failed with:

```text
Static check failed: guided-workflow contract missing workflow guidance panel exists
```

The workflow guidance panel existed and remained behaviorally valid, but its attributes were ordered in a way that did not satisfy the existing literal static token.

## Change

Reordered the `#workflowGuidance` attributes so the static contract can find the exact expected token:

```html
id="workflowGuidance" data-guidance-state="topic" data-guidance="active"
```

## Non-goals

- No runtime logic change.
- No visual design change.
- No CSS behavior change.
- No import/export/schema behavior change.
- No browser selector change.

## Files changed

```text
index.html
docs/design/xr-13.0.3-guided-workflow-contract-token-hotfix.md
```
