# v1.0.6 — Documentation + Release Packaging Cleanup

## Scope

v1.0.6 is a patch-only documentation and release packaging cleanup. It adds no new provider behavior, OAuth behavior, backend endpoint behavior, source connector behavior, storage model, or schema-breaking workflow capability.

## What changed

- Corrected historical release labels that had drifted to v1.0.5 in README, changelog, QA matrix, and versioned docs.
- Added `RELEASE_MANIFEST.md` as the canonical package inventory.
- Added `.releaseignore` to document files excluded from release archives.
- Removed the duplicate/misnamed v1.0.5 browser-QA documentation file.
- Added v1.0.6 migration and privacy snapshots.
- Added `tests/release-packaging-cleanup-check.mjs`.
- Added `tests/v106-no-browser-suite.mjs`.

## Compatibility guard

This release is disproven if any of the following happens:

- Provider mode behavior changes.
- OAuth/token behavior changes.
- Backend endpoints change.
- Source connector behavior changes.
- A schema-breaking workflow change appears without migration.
- Release docs reintroduce historical v1.0.5 label drift.
- Release archive hygiene allows generated/local artifacts into the package.

## Validation

```bash
npm run test:release-packaging
npm run test:v106:no-browser
npm run test:ci:no-browser
```

Browser validation remains required before publishing:

```bash
npm run test:ci:browser
```
