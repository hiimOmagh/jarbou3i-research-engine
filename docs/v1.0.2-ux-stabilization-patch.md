# v1.0.3 — UX Stabilization Patch

## Purpose

v1.0.3 is a patch-only UX stabilization release. It does not add research-engine capability, source connectors, provider behavior, OAuth behavior, or schema-breaking workflow changes.

The goal is to reduce the visible operational wall exposed in v1.0.1 and make the stable product workflow legible on first load.

## Added

- Top-level workflow navigation tabs:
  - Analysis
  - Evidence
  - Sources
  - Quality & Export
  - Settings / Advanced
- Release Health summary card.
- Collapsed advanced panels by default.
- UX stabilization regression test.
- v1.0.1 → v1.0.3 migration fixture.
- v1.0.3 privacy fixture.

## UX rule

The first screen must answer:

```text
What is this tool?
What am I analyzing?
What is the next action?
Is the analysis safe/exportable?
Where is my evidence?
```

## Non-goals

- no new source connector
- no live web search
- no production OAuth enablement
- no new provider behavior
- no privacy/export weakening
