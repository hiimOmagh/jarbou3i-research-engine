# v1.0.5 — Screen Discipline Patch

## Purpose

v1.0.5 is a patch-only UX discipline release. It does not add engine capability, provider behavior, OAuth behavior, source connectors, or schema-breaking features.

The goal is to reduce screen-level noise after the v1.0.2 workflow navigation patch.

## Changes

- Removed global section numbering from research cards.
- Collapsed Analysis Command Center by default.
- Collapsed Analysis Engine Map by default.
- Added explicit show/hide controls for Command Center and Engine Map.
- Added compact next-action guidance to the Release Health card.
- Compressed empty states.
- Converted advanced/internal panels into accordion-style collapsed cards.
- Kept workflow tabs from v1.0.2 intact.

## Product rule

The first screen must now emphasize one path:

```text
Plan → Evidence → Quality → Export
```

Advanced/provider/OAuth/backend/source internals remain available, but they no longer compete visually with the primary workflow by default.

## Non-goals

- No new provider.
- No new connector.
- No production OAuth enablement.
- No new live search behavior.
- No Evidence Matrix bypass.
- No privacy/export guard weakening.
