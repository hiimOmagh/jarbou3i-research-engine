# v0.21.0-beta — Project Workspace + Local Storage Management

## Purpose

This release adds a local-only project workspace for managing multiple research packets without introducing cloud persistence, accounts, or live integrations.

## Added

- `src/research/project-workspace.js`
- Named local project saves
- Project duplicate/delete/export/import controls
- Local storage diagnostics
- `project_workspace` metadata in exported research packets
- v0.21 migration support from v0.11-v0.20 packets
- `tests/project-workspace-check.mjs`
- `tests/v021-no-browser-suite.mjs`

## Guardrails

- Storage is explicitly browser-local only.
- No cloud persistence was added.
- No account system was added.
- No real OAuth was added.
- Project export still passes through the privacy export guard and final privacy audit.
- Raw API keys and OAuth tokens remain forbidden in exports.

## Acceptance

- Users can save, duplicate, delete, export, and import named local projects.
- Workspace metadata is exported for auditability.
- Privacy audit and migration checks remain release gates.
- Manual/private mode remains first-class.
