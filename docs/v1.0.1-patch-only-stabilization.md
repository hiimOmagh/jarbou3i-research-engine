# v1.0.4 — Patch-only Stabilization

## Scope

v1.0.4 is a stabilization patch on top of v1.0.0. It does not add new product capabilities, connectors, OAuth behavior, provider behavior, or schema-breaking workflow changes.

## Fixed

- Corrected localized stable-release badge copy in `src/research/render-helpers.js`; v1.0.0 still showed RC wording in rendered language-copy strings.
- Updated CI no-browser syntax gate to check the active stable no-browser suite instead of the obsolete RC wrapper.
- Added `tests/patch-stabilization-check.mjs` to enforce patch-only stabilization constraints.
- Added `tests/v102-no-browser-suite.mjs`.
- Added v1.0.0 → v1.0.4 migration fixture support.
- Added v1.0.4 privacy fixture.

## Patch policy

Allowed:

```text
bugfix
docs
qa
a11y
privacy_audit
migration_compatibility
release_packaging
patch_release
```

Blocked without a new minor version and threat model:

```text
new major features
new live providers
new source connectors
production OAuth enablement
schema-breaking changes without migration
secret export weakening
```
