# v0.29.0-rc.1 — Export Pack v2

## Purpose

Export Pack v2 turns a research workflow into a professional handoff bundle instead of a single opaque JSON file.

The bundle is designed for:

- GitHub archival
- Claude / ChatGPT handoff
- local research archive
- publication pipeline preparation
- privacy/security audit trails

## Export artifacts

The pack generates these artifacts as separate browser downloads:

```text
export-manifest.json
research-packet.json
analysis-brief.md
evidence-matrix.csv
review-queue.csv
provider-run-ledger.json
quality-report.json
privacy-audit.json
```

The original planned core bundle remains intact:

```text
research-packet.json
analysis-brief.md
evidence-matrix.csv
review-queue.csv
provider-run-ledger.json
quality-report.json
privacy-audit.json
```

`export-manifest.json` is added as a machine-readable index for file names, MIME types, byte counts, checksums, topic, quality readiness, and privacy release-gate status.

## Privacy model

Each artifact is produced through the existing privacy-safe export boundary.

The pipeline is:

```text
research packet
→ privacy export guard
→ privacy audit final pass
→ export pack artifact generation
→ per-file privacy scan
→ download
```

Forbidden raw provider keys, bearer tokens, OAuth access tokens, refresh tokens, and secret-shaped values must be removed or redacted before artifacts are downloaded.

## Workflow metadata

Research packets now include:

```json
{
  "export_pack": {
    "export_pack_version": "0.29.0-rc.1",
    "format": "export_pack_v2",
    "files": [
      "research-packet.json",
      "analysis-brief.md",
      "evidence-matrix.csv",
      "review-queue.csv",
      "provider-run-ledger.json",
      "quality-report.json",
      "privacy-audit.json"
    ],
    "release_gate": "privacy_audit_required"
  }
}
```

## Migration

v0.23 packets migrate to v0.24 with safe `export_pack` defaults.

Migration does not introduce live provider behavior, real OAuth, source fetching, or secret storage.

## Guardrails

- Manual/private mode remains first-class.
- Export Pack v2 is a packaging layer, not a live integration layer.
- Evidence Review Queue remains the source-import gate.
- Provider run ledgers remain privacy-scanned.
- Quality reports are exported as diagnostic artifacts, not as proof of truth.
